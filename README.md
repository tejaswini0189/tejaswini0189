## 📌 Featured Projects

### 🤚 Gesture-Based Nurse Assistance System
> *Bridging the gap between patients and caregivers through computer vision*

A real-time gesture recognition system that empowers patients to communicate without verbal interaction. When a patient performs a gesture in front of a webcam, the system instantly sends a notification alert — complete with an identification sound — to the attending nurse or doctor, enabling faster response times and better patient care.

**Tech Stack:** Python · OpenCV · Computer Vision · Real-time Notifications

---

### 🌾 Crop Cart
> *Empowering farmers through direct market access*

An agricultural marketplace platform that directly connects farmers with dealers, eliminating middlemen and enabling fair trade. Dealers can pre-book crop stock based on real-time availability, giving farmers a reliable market and dealers a predictable supply chain — a win-win for both sides of the agricultural ecosystem.

**Tech Stack:** Python · Web Development · Database Management
